1. Download the data pack from a link above. It should be a .zip file.
  2. Unzip the file.
  3. Open Minecraft.

    4. Data Pack

  4. Select the world you want to install the data pack for, click on "Edit", then "Open world folder".
  5. Open the folder named datapacks, and drag ttb_data from the unzipped folder into it.
  6. Enter the world.
  7. Type /reload in the chat. If you do not have cheats enabled, press Esc, click on "Open to LAN", set "Cheats" to "Enabled", and confirm.
  8. After a few seconds, you should see a message in chat saying The Tempest's Box: Loaded, confirming that the pack is installed.

    9. Resource Pack

  9. From the main menu, click on "Options", then "Resource packs", then "Open resource pack folder".
  10. Drag ttb_resources from the unzipped folder into it.
  11. Click "Done" in the resource packs menu, then go back into it.
  12. Enable the resource pack named ttb_resources.
